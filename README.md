# Quest Craft

A video game built by kids, for kids — on a Raspberry Pi 5.

## About

This game is being built by a team of young developers (ages 9-11) for the **LASD Illuminate Inquiry Exhibition** on **March 15, 2026**.

**Built with:**
- Python 3.13 + Pygame 2.6
- Raspberry Pi 5
- Nintendo Switch controllers
- Claude Code (AI coding partner)
