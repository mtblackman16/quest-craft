Key Insights Reflect
Eins
"How fast the AI can develop a single entire game, like all the music and stuff, without us asking it too hard."
Ethan
"You need context. If you just say 'make a game', then it wouldn't know what game to make."
Andrew
"I should not worry so much about how the designs that I make turn out, and that I should just go with the flow."
Nathan
"Even if the Raspberry Pi breaks, you can use GitHub to still have the code."
