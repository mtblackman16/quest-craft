Hard Question Answers

"Did the AI do all the work?"
We designed everything — characters, levels, bosses, enemies, controls. Claude wrote the code from our designs. The hardest part was knowing what to ask for.
"Did you write any code?"
We directed the code. We told Claude what to build, tested it, found bugs, and told it what to fix. We learned about assets, folders, Git, and how the code is organized.
"How is this different from ChatGPT?"
Claude Code lives inside our project. It reads our files, writes real code, and runs the game. We talked to it like a teammate — 5 sessions over a month.
